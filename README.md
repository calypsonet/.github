# calypsonet/.github

This repository holds organization-wide GitHub configuration and operational documentation
for the [`calypsonet`](https://github.com/calypsonet) organization.

---

## Contents

### Community Health Files

Default community health files that apply to all `calypsonet` repositories that do not
provide their own:

| File | Purpose |
|---|---|
| `CODE_OF_CONDUCT.md` | Community standards and enforcement guidelines |
| `CONTRIBUTING.md` | How to contribute to calypsonet projects |
| `SECURITY.md` | How to report security vulnerabilities |
| `SUPPORT.md` | How to get help |

### Workflow Templates

Reusable workflow templates available to all repositories in the organization are maintained
in the [`calypsonet/calypsonet-actions`](https://github.com/calypsonet/calypsonet-actions)
repository.

---

## Operational Documentation

The [Wiki](https://github.com/calypsonet/.github/wiki) of this repository contains
operational procedures for maintainers of the `calypsonet` organization:

- **[Credential Management](https://github.com/calypsonet/.github/wiki/Organization-Secrets)** — registering and rotating organization secrets, GPG keys, Maven Central tokens, and GitHub PATs
- **[Maven Central Publication](https://github.com/calypsonet/.github/wiki/Maven-Central-Overview)** — publishing JVM libraries under `org.calypsonet.keyple`
- **[Troubleshooting](https://github.com/calypsonet/.github/wiki/Maven-Central-Common-Errors)** — common CI/CD errors and corrective actions

---

## Organization

The `calypsonet` organization develops open-source libraries for the Calypso ecosystem,
including Keyple plugins for smart card communication on various platforms and hardware.

- Main project: [Keyple](https://keyple.org)
- Reference API: [Terminal API](https://terminal-api.calypsonet.org)
